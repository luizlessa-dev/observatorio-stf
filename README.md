# Observatório do STF

Plataforma de acompanhamento e análise das decisões do **Supremo Tribunal Federal** (STF), tornando a jurisprudência acessível e compreensível para cidadãos, jornalistas e pesquisadores.

## Objetivo

O Observatório do STF monitora, categoriza e apresenta as decisões do Supremo de forma clara e estruturada, permitindo que qualquer pessoa entenda o impacto das decisões judiciais no cotidiano brasileiro.

## Funcionalidades

- Acompanhamento de decisões em tempo real
- Categorização por tema (direitos fundamentais, questões eleitorais, economia, etc.)
- Linha do tempo de casos relevantes
- Análise de impacto e repercussão das decisões

## Tecnologia

- Stack a definir (projeto em desenvolvimento)
- Dados: API pública do STF / scraping de fontes abertas

## Contexto

Este projeto faz parte de um conjunto de iniciativas de dados cívicos e transparência pública, junto com a [Transparência Federal](https://github.com/luizlessa-dev/transparencia-federal).
